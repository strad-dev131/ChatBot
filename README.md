## 💻 **Installation & Deployment**

### **Method 1: Local Development**
```bash
# Clone repository
git clone https://github.com/strd-dev131/ChatBot.git
cd ChatBot

# Install requirements  
pip install -r requirements.txt

# Setup environment
cp Sample.env .env
nano .env  # Add your credentials

# Run the bot
python main.py
```

### **Method 2: VPS Deployment (24/7)**
```bash
# Ubuntu/Debian
sudo apt update && sudo apt upgrade -y
sudo apt install python3 python3-pip git nodejs npm -y

# Clone and setup
git clone https://github.com/strd-dev131/ChatBot.git
cd ChatBot
pip3 install -r requirements.txt

# Configure environment
cp Sample.env .env
nano .env  # Add your variables

# Install PM2 for 24/7 operation
sudo npm install -g pm2

# Start bot with PM2
pm2 start main.py --name "EnaChatBot" --interpreter python3

# Save PM2 configuration for auto-restart
pm2 save
pm2 startup
```

### **Method 3: Docker**
```bash
# Clone repository
git clone https://github.com/strd-dev131/ChatBot.git
cd ChatBot

# Build and run
docker build -t ena-chatbot .
docker run -d --env-file .env ena-chatbot
```

### **Method 4: Heroku (One-Click)**
1. Click the "Deploy to Heroku" button above
2. Fill in the required environment variables
